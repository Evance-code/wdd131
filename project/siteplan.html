W05 Project: Website Planning Document

Tanzania JobConnect

Site Name

Tanzania JobConnect — connecting Tanzanians with verified employment opportunities.
Optional Domain: tanzaniajobconnect.org

Site Purpose

Provides a platform for Tanzanians to discover jobs, submit applications, and access career resources nationwide.

Scenarios

  • Find jobs in Dar es Salaam matching my qualifications.
  • Submit applications online directly to employers.
  • Verify job listings for authenticity.

Color Scheme

Tanzanian flag inspired colors: green for headers/nav/footer, yellow for buttons, blue for accents.

Typography

  • Font: Open Sans
  • Headers: bold, green
  • Body text: regular, black

Wireframe (Mobile)

Nav → Hero → Job Highlights → Footer

Wireframe (Desktop)

Header + Horizontal Nav → Hero Section → 3-Column Features → Footer
